# 대규모 트래픽 처리에 대한 성능개선을 위한 프로젝트
##  ⚙️개발환경
- Frontend: React, Axios, React Router, Bootstrap
- Backend: Spring Boot, Spring MVC, Spring Security, Spring Data JPA
- API 설계: RESTful API, JSON 기반 통신
- DB: MySQL
- ORM: JPA
- 인증/인가: Spring Security, JWT 기반 인증
- 빌드 도구: Gradle
